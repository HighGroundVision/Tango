The Artifact Puzzle Builder

Every artifact player has played at least one puzzle even if they did not know it. The tutorial, or the the 2 forced matches when you first started playing artifact where set up as puzzles (In fact you can recreate that tutorial with this tool). A puzzle is a txt file that is loaded via the console in artifact where the player has a match against an AI where there is set broad state and set of actions for each turn for the AI.

OpenDota

OpenDota is a volunteer-developed, open source platform providing Dota 2 data. It provides a web interface for casual users to browse through the collected data, as well as an API to allow developers to build their own applications with it.
