Path Notes: 7.07

  • New Drafting UI
  • Added support for chat during draft.
  • Added support for swapping ability slots.
  • Added tooltips for hero talents and base stats as well as indicators for hero primary attributes.
  • Scoreboard will now show drafted abilities at end of game and when viewing replays.
  • 17 New heroes. Including Dark Willow and Pangolier.
  • 6 Existing heroes have been fixed.

What dose this mean for AD?

  • It is nice to see AD getting some love.
  • This change will fix a number of existing game breaking bugs.
  • As well as introduce some new heros and some old favourites.
  • Although these heroes will have limited abilities in the pool.
  • Hopefuly this will translate into more people playing Ability Draft!

What dose this mean for HGV?

A number of these changes has maybe the existing Ability Drafter website obsolete.

  • Tooltips for base stats
  • Tooltips for primary attribute
  • Tooltips for talents

Since the primary focus of the website was to provide these details to drafters and that will no longer be needed after the update we are refocusing the website towards drafting.

While we had this feature in the existing website it never worked correctly because most of the data was infered from the ability level up data which was often incomplete and did not include the drafting order.

With the addation of the drafted abilities in the Scoreboard we now have the data we need to do a better job!

Infact the Crystalys & Orchid Projects where built to get the Scoreboard details. Althought they need to be updated to use the new protobuff files.

Next Steps
